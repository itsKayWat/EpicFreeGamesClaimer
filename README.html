EpicFreeGamesClaimer

About

EpicFreeGamesClaimer is an automated tool that helps you claim free games from the Epic Games Store without having to manually check and claim them each week.

Why Use This?

  • Never miss free games again
  • Automatically claims games while you're busy
  • Perfect for users who forget to check weekly free games
  • Build your game library effortlessly

Requirements

  • Python 3.7+
  • Chrome Browser
  • Chrome WebDriver
  • Internet connection

Installation

  1. Clone this repository
  2. Run: python requirements.py
  3. Configure your Epic Games credentials in config.ini

Usage

python epicgames-claimer.py

Use Case Example

John works long hours and often misses the weekly free games on Epic Games Store. By running this script weekly (or setting up a scheduled task), he never misses a free game, building his game library automatically.
